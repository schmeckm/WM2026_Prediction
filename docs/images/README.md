## How to Capture

1. Start the app locally (`npm run dev` in `frontend/` and `backend/`)
2. Log in with the demo account (`max.mueller@example.com` / `user123`)
3. Take screenshots in the browser (e.g. Windows: `Win + Shift + S`)
4. Save as PNG in this folder
5. Update `README.md` to reference `.png` instead of `.svg` (if still linked as SVG)
